Книги

Основные работы для ознакомления с подходом

Domain-Driven Design

Tackling Complexity in the Heart of Software

Eric Evans

Большая Синяя Книга, которая заложила основы методологии предметно-ориентированного программирования.

Domain-Driven Design Reference

Eric Evans

Книга представляет собой краткий обзор методологии DDD. В ней также приведены определения терминов, которые появились в теме с момента публикации Большой Синей Книги в 2003 году.

Implementing Domain-Driven Design

Vaughn Vernon

Большая Красная Книга рассказывает, как применить DDD на практике. Основываясь на книге Эрика Эванса, автор описывает техники предметно-ориентированного программирования на примерах из общеизвестных доменов.

Предметно-ориентированное программирование

Вон Вернон

Если вы только начинаете изучать предметно-ориентированное проектирование, то рекомендуем начинать с этой книги и DDD Reference от Эванса. Так будет много проще входить в тему.

Introducing EventStorming

An act of deliberate collective learning

Alberto Brandolini

Самое основательное пояснение и учебное пособие по EventStorming, прямо от создателя техники. Книга обучает как использовать потенциал этой методики совместного мозгового штурма для определения событий предметной области.
