Библиотеки и фреймворки

Инструменты, помогающие с DDD в коде

  • Spine Event Engine
    CQRS/ES фреймворк с открытым исходным кодом, помогающий создавать облачные приложения на основе DDD.
  • Axon
    Фреймфорк с открытым исходным кодом для микросервисов, основанных на событиях, и для предметно-ориентированного программирования.
  • Vlingo
    Набор инструментов от Вона Вернона для реактивных архитектур, основанных на событиях и микросервисах.
  • EventStore
    Потоковая база данных от Грега Янга, построенная для Event Sourcing.
